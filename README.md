# Getting Started

## Prerequisites

npm

`npm install npm@latest -g`

## Installation

1. Clone the repo
    `git clone https://github.com/Vicky10909/InvestmentCalculator.git`
2. Install NPM packages
   `npm install`
3. Run the app on a local server
   `npm run dev`
